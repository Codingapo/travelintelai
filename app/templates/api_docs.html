TravelIntel REST API

Integrate real-time booking intelligence into your agency's platform.

🔑 Authentication

All API requests must be authenticated using your agency's unique API key. Include it in the x-api-key header.

x-api-key: {{ request.session.get('agency_api_key', 'your_api_key_here') }}
📡 Endpoints
POST /events

Send a booking event to TravelIntel for real-time evaluation, anomaly detection, and forecasting.

Request Body (JSON)

{
  "event_id": "evt-123456789",
  "event_type": "booking.created",
  "sent_at": "2026-03-19T12:00:00Z",
  "data": {
    "booking_id": "BK-9988",
    "customer": {
      "name": "Jane Doe",
      "email": "jane@example.com"
    },
    "trip": {
      "destination": "Cape Town",
      "departure_date": "2026-04-10"
    },
    "payment": {
      "currency": "ZAR",
      "amount": 8500
    }
  }
}
GET /bookings/{booking_id}

Retrieve the full details and AI evaluation results for a specific booking.

💻 Code Examples
PHP (cURL) 
$ch = curl_init('https://api.travelintel.ai/events');
$payload = json_encode([
    'event_id' => 'evt-' . bin2hex(random_bytes(8)),
    'event_type' => 'booking.created',
    'sent_at' => gmdate('c'),
    'data' => [
        'booking_id' => 'BK-1234',
        'customer' => ['name' => 'Jane Doe', 'email' => 'jane@example.com'],
        'trip' => ['destination' => 'Cape Town', 'departure_date' => '2026-04-10'],
        'payment' => ['currency' => 'ZAR', 'amount' => 8500]
    ]
]);

curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $payload);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'x-api-key: YOUR_API_KEY'
]);

$response = curl_exec($ch);
curl_close($ch);
echo $response;
Python (Requests) 
import requests
import uuid
import datetime

url = "https://api.travelintel.ai/events"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
payload = {
    "event_id": f"evt-{uuid.uuid4().hex[:16]}",
    "event_type": "booking.created",
    "sent_at": datetime.datetime.utcnow().isoformat() + "Z",
    "data": {
        "booking_id": "BK-1234",
        "customer": {"name": "Jane Doe", "email": "jane@example.com"},
        "trip": {"destination": "Cape Town", "departure_date": "2026-04-10"},
        "payment": {"currency": "ZAR", "amount": 8500}
    }
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
