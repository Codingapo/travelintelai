Integration Guide

Up and running in 3 steps

No complex setup. Your agency can be live in under 10 minutes.

01

Register Your Agency

Create your free account. You'll instantly receive a unique API key that authenticates all your booking events with TravelIntel.

02

Send Booking Events

When a customer books, POST the booking payload to our /api/v1/events endpoint. We evaluate, store, and process it in milliseconds.

03

Monitor Your Dashboard

Log in to your agency dashboard to see real-time analytics, AI forecasts, customer segments, and all booking activity in one place.

PHP · travelintel_client.php Example Integration 
$payload = [
  'event_id'   => 'evt-' . bin2hex(random_bytes(8)),
  'event_type' => 'booking.created',
  'sent_at'    => gmdate('c'),
  'data'       => [
    'booking_id' => 'BK-1234',
    'customer'   => ['name' => 'Jane Doe', 'email' => 'jane@example.com'],
    'trip'       => ['destination' => 'Cape Town', 'departure_date' => '2026-04-10'],
    'payment'    => ['currency' => 'ZAR', 'amount' => 8500],
  ],
];
travelintel_post('/events', $payload);
